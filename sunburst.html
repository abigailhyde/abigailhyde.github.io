"Sunburst Chart: Cremation vs. Burial and Other Disposition Techniques"

"Sunburst Chart: Cremation vs. Burial and Other Disposition Techniques" was my final project for CS 490R at BYU-Hawaii in Winter 2022. This visualization displays cremation statistics for five countries, making it easy to compare trends over time. The dataset and code can be found here. This program was written in Java, and made extensive use of JDBC.
